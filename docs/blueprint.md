# **App Name**: LiFE-iN-SYNC

## Core Features:

- Daily Habit Tracker: Log daily habits with customizable reminders. Displays user's current streaks.
- AI-Powered Insights: Generative AI tool interprets journaling entries to provide insights on user's mood, thoughts, and behaviors. User is given daily prompts to provoke deep thought self reflection.
- Finance Module: Set personal budgets and visualize progress through charts and graphs.
- AI Knox: No nonsense , call it like it is AI Therapist who will always give honesty and harsh truth needed for healing and growth , over comfort and avoidance.
- Workout Module: Construct templates with exercises than will be complied into a timed organized workout session.
- Tasks Module: Log daily objectives that are essentially little reminders that I can quickly add throughout the day and set priority levels.
- Shopping List: Write down grocery and other apartment essentials that need to be restocked
- Calendar Module: Interactive calendar with customizable dates to log events and plans throughout the year

## Style Guidelines:

- Primary color: Deep burgundy (#800020), reflecting sophistication and focus.
- Background color: Dark grey (#212121), for a neumorphic, visually subdued, dark mode interface.
- Accent color: Mauve (#E0B0FF), for highlights and interactive elements, ensuring clear affordances against the dark background.
- Body and headline font: 'Inter', a versatile sans-serif suitable for both headlines and body text.
- Minimalist icons with subtle shading and depth, aligning with the neumorphic UI.
- Neumorphic design with soft shadows and rounded edges for a tactile feel.
- Smooth transitions and subtle animations for a polished user experience.
